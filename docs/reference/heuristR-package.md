# heuristR: Session-Oriented Access to Heurist

`heuristR` provides a session-oriented wrapper around Heurist
authentication, metadata, and record retrieval endpoints. The package is
designed to support safe client-side merge workflows for record updates,
rather than relying on destructive partial save calls.
